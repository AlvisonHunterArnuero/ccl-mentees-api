Users API Documentation

This guide covers the endpoints for managing users in the CCL Mentoring Program API.

Create User

Use this endpoint to create a new user.

URL

POST /users

Request Body

{
    "name": "string",
    "password": "string",
    "role": "admin" | "guest"
}

Response

{
    "message": "User created successfully"
}

Get All Users

Use this endpoint to retrieve all users.

URL

GET /users

Response

[
    {
        "id": 1,
        "name": "string",
        "password": "string",
        "role": "admin" | "guest"
    },
    ...
]

Get User by ID

Use this endpoint to retrieve a user by their ID.

URL

GET /users/<id>

Response

{
    "id": 1,
    "name": "string",
    "role": "admin" | "guest"
}

Update User

Use this endpoint to update an existing user's information.

URL

PUT /users/<id>

Request Body

{
    "name": "string",
    "password": "string",
    "role": "admin" | "guest"
}

Response

{
    "message": "User updated successfully"
}

Delete User

Use this endpoint to delete a user.

URL

DELETE /users/<id>

Response

{
    "message": "User deleted successfully"
}
