Login API Documentation

This guide covers the authentication endpoints for logging in and out of the CCL Mentoring Program API.

Login Endpoint

Use this endpoint to log in to the API.

URL

POST /login

Request Body

{
    "name": "string",
    "password": "string"
}

Response

{
    "message": "Logged in successfully"
}

Error Response

{
    "message": "Invalid credentials"
}

Logout Endpoint

Use this endpoint to log out of the API.

URL

GET /logout

Response

{
    "message": "Logged out successfully"
}
