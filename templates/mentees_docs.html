Mentees API Documentation

This guide covers the endpoints for managing mentees in the CCL Mentoring Program API.

Create Mentee

Use this endpoint to create a new mentee.

URL

POST /mentees

Request Body

{
    "name": "string",
    "last_name": "string",
    "email": "string",
    "phone": "string",
    "english_level": "string",
    "python_level": "string",
    "javascript_level": "string",
    "seniority": "string",
    "linkedin_profile": "string",
    "github_profile": "string",
    "website": "string",
    "final_grade": 0.0
}

Response

{
    "message": "Mentee created successfully"
}

Update Mentee

Use this endpoint to update an existing mentee's information.

URL

Request Body

{
    "name": "string",
    "last_name": "string",
    "email": "string",
    "phone": "string",
    "english_level": "string",
    "python_level": "string",
    "javascript_level": "string",
    "seniority": "string",
    "linkedin_profile": "string",
    "github_profile": "string",
    "website": "string",
    "final_grade": 0.0
}

Response

{
    "message": "Mentee updated successfully"
}

Delete Mentee

Use this endpoint to delete a mentee.

URL

DELETE /mentees/<id>

Response

{
    "message": "Mentee deleted successfully"
